`Signal Deck` is a standalone browser-based mental math and quant interview practice site built as a flashcard game. Users can choose topics, difficulty, timer, and session length, then work through approximation-heavy problems on logs, roots, probability, random walks, expected values, covariance/correlation, Green Book classics, and other trader-style quant drills, with approximate-answer grading, live scoring, and end-of-session review.
